Here's a list of commands provided by the bot:

- **/start**: Begin interaction with the bot (authorizes the user).

- **/help**: Display the help guide.

- **/status**: Check if the bot is running successfully.

- **/save**: Download bot data (JSON files and logs).

- **/allow [user_id]**: Admins can authorize a user.

- **/deny [user_id]**: Admins can deny a user's authorization.

- **/make_admin [user_id]**: Creator can promote a user to admin status.

- **/del_admin [user_id]**: Creator can remove admin status from a user.

- **/ban [user_id]**: Admins can ban a user.

- **/free [user_id]**: Admins can unban a user.

- **/delete [user_id]**: Admins can delete a user.

- **/text [message]**: Admins can send a message to all users.

- **/text_to [user_id] [message]**: Admins can send a message to a specific user.

- **/again**: Re-run a task, if available.

- **/show_troubles**: Show issues, if any.

- **/clear**: Clear previous tasks.

- **/users**: Show the list of authorized users.

- **/all**: Show all registered nicknames.

- **/banned**: Show the list of banned users.
